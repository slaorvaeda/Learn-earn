## âš¡ Apache Kafka Overview

### What is Kafka?
- **Distributed event streaming platform**
- **High-throughput** message broker
- **Real-time** data processing
- **Event-driven** architecture

### Key Concepts
- **Topics**: Categories for messages
- **Producers**: Send messages
- **Consumers**: Read messages
- **Brokers**: Servers in cluster
- **Partitions**: Parallel processing

## ğŸš€ Quick Start

### 1. Install Node.js
```bash
# Check version
node --version  # v18+

# Install from nodejs.org
```

### 2. Install MongoDB
```bash
# macOS
brew install mongodb-community

# Start MongoDB
brew services start mongodb-community
```

### 3. Install Kafka
```bash
# Download from kafka.apache.org
# Extract and start Zookeeper
bin/zookeeper-server-start.sh config/zookeeper.properties

# Start Kafka
bin/kafka-server-start.sh config/server.properties
```

### 4. Create Your First App
```bash
# Create project
mkdir my-mern-kafka-app
cd my-mern-kafka-app

# Initialize backend
mkdir backend && cd backend
npm init -y
npm install express mongoose dotenv cors

# Initialize frontend
cd .. && npx create-react-app frontend
```
